Wat is html en css?

Het vak HTML&CSS is een van de basis vakken die wij het eerste jaar krijgen. onze docent Andr√© Pape geeft ons les in het vak. dit vak is het eerste codeer vak dat wij krijgen op onze opleiding applicatie ontwikkelaar maar ook ICT beheer krijgt hier het eerste jaar les in. HTML en CSS zijn code talen die je kan gebruiken om website te bouwen en optemaken. HTML(HyperText Markup Language) is voor de content op de website en CSS (Cascading Style Sheets) is algemeen voor de opmaak van de website. HTML kun je eigenlijk zien als een blokkendoos waarmee alle onderdelen van de website zoals de "Footer"of een "header" eigenlijk een block is en waar je dan allemaal dingen in kan zetten en met css kun je die blocken bijvoorbeeld groter maken, een rand geven of ergens anders plaatsen, met css kun je bijvoorbeeld ook het lettertype een andere kleur geven.


Waarom we voor HTML&CSS hebben gekozen?

Ookal hebben we zoveel leuke vakken hadden we toch maar gekozen voor het vak HTML&CSS. Deze reden hiervoor is dat we het leuk vinden om te coderen en om websites te maken. we kunnen mooi onze creativiteit in weg stoppen. Maar we vinden ook dat dit voor applicatie een belangerijk vak is om te leren omdat deze talen wel de basistalen zijn en dat we deze wel moeten kennen als we later front-end developers zouden worden.
