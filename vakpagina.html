HTML&CSS

Samenvatting

Het vak html&css kan iedereen. HTML is de opmaak van een website. Dit is de voorkant die je ziet van een website als je een website opzoekt. Samen met CSS erbij zorgt het ervoor dat je website er mooi uitziet. Wat wij tot nu toe hebben gehad met html&css valt ook op deze website te zien.

Het begon allemaal met simpele dingen zoals de kleur van tekst veranderen. Dit groeide langzaam uit naar een werkende website. De website zag er uiteindelijk ook uit als een website. Onze aangeleerde html&css-skills is verwerkt in deze website. We hopen zo een goede indruk te geven van wat we allemaal al hebben geleerd met HTML&CSS.
